## 🚀 Projects I'm Proud Of

Full lab simulating a dynamic spoke-to-spoke VPN topology using FortiGate.
Built to understand how enterprise networks are designed — and where they break.

Interactive mini-blockchain with and without Proof-of-Work, including a live 51% attack scenario.
